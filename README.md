A custom game engine in the making:
- SDL2 for windowing, events
- Opengl 4.1 renderer
- Dear ImGUI for GUI
- premake (configured for visual studio only right now)
- Executable folder needs to have the following for the engine to run:
    - res/ directory
    - scene.graph file
    - SDL2.dll
